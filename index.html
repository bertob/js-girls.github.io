About

JS Girls is a 1-day workshop dedicated to girls and women willing to learn and enjoy one of the most used language for the web.

JS Girls aims to involve more female developers within the Javascript community. You will learn basics of programming and get introduced to the world of today's Javascript technology. JS Girls is a joint initiative of Milano JS and Girls in Tech Italy.
